# Clockie
Clockie is the best chess clock for the iPhone.

# Status
Free on [the App Store](https://itunes.apple.com/us/app/clockie-minimal-chess-clock/id816844998?ls=1&mt=8).

# Features 
- It's orange 
- Sub-millisecond precision 
- Pause and resume 
- It's got settings 
- No ads 
- Free!
